# Setup Rápido - PIX Recorrente

## Para rodar NO DISPOSITIVO FÍSICO:

1. Conecte o celular no USB
2. Ative Depuração USB no celular
3. Execute:

```bash
yarn android
```

Pronto! O app vai abrir no seu celular.
